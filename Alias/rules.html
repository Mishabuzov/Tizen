Alias или Скажи иначе - словестная командная игра, в которой один игрок из команды должен объяснить как можно больше слов, показанных на экране, своим партнерам. При объяснении нельзя использовать однокоренные слова, переводы на иностранные языки и явно показывать слово жестами.


Начисление очков

За отгаданное слово команда получает 1 очко. За пропущенное -1. Если команда не успела ответить на последнее слово - она получает 0 очков.


Определение победителя

Игра длится до тех пор, пока одна из команд не наберёт необходимое для победы количество очков, которое задаётся перед началом игры.
